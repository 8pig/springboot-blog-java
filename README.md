###### 开发前的配置要求

1. Java 1.8.0_212
2. Maven 3.8.2
3. Mysql 5.7.26
4. IDEA 2021.2.3
5. Webstorm 2021.2.3
6. node 14.17.6
7. npm 7.21.1
8. mybatis-plus 3.4.3
9. redis 6.0.6

###### **安装步骤**

1. clone this reop
2. Import Project `pom.xml`
3. Run the database file (*.sql)
4. `application.properties` Modify database links
5. run BlogApp

```sh
git clone https://github.com/8pig/forJava.git
```
